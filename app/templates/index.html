OneLife is our solution to suicide. It is a web app and twitter bot that uses advanced machine learning algorithms to identify suicidal messages from social media chats.

Try it out!

Why OneLife?

We use comprehensive machine learning algorithms and natural language processing to analyze text messages and classify them based on our bayesian filter, trained from a reddit dataset that we scraped. Our bot makes use of the twitter API to message motivational texts and helpline numbers to the suicidal individual. It also enables users to either copy-paste the raw text of the message, or simply upload an image of the note/text. we then extract the text from this image and provide seamless results.
